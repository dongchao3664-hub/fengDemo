# 海缆高程分析功能说明

## 功能概述

为海缆数据新增了高程分析功能，通过 ECharts 图表展示海缆各测量点的高程数据，并支持预警线设置和数据导出。

## 主要特性

### 1. 高程分析图表
- **单条线段模式**：展示当前选中海缆的所有测量点高程
- **全部数据模式**：展示所有海缆的测量点高程（散点图）
- **预警线**：可调整的预警阈值线，默认 -2.0 米
- **交互式图表**：支持缩放、拖拽、悬浮显示详细信息

### 2. 触发方式

#### 方式一：信息面板触发
1. 在左侧海缆列表中选择一条海缆
2. 右侧信息面板会显示海缆详情
3. 点击"高程分析"按钮打开图表

#### 方式二：列表快捷按钮
1. 在左侧海缆列表中，每条海缆右侧有图表图标按钮
2. 直接点击图标按钮即可打开该海缆的高程分析图表

#### 方式三：点击地图线缆
1. 在地图上直接点击海缆线段
2. 自动选中该海缆并打开信息面板
3. 在信息面板中点击"高程分析"按钮

### 3. 图表功能

#### 视图切换
- **当前线段**：展示选中海缆的折线图，清晰显示高程变化趋势
- **全部数据**：展示所有海缆的散点图，便于对比分析

#### 预警阈值
- 通过输入框调整预警线高度（单位：米）
- 在"全部数据"模式下，低于预警线的点会显示为红色
- 接近预警线的点显示为橙色
- 正常点显示为绿色

#### 数据导出
- 点击"导出数据"按钮
- 自动生成 CSV 格式文件
- 包含海缆名称、点位名称、经纬度、高程等信息

## 图表类型说明

### 单条线段模式（折线图）
- **X轴**：测量点名称
- **Y轴**：高程（米）
- **展示方式**：折线图 + 面积填充
- **颜色**：根据海缆类型（掩埋=绿色，裸露=橙色）
- **交互**：
  - 悬浮显示点位详细信息
  - 支持缩放和拖拽
  - 预警线以红色虚线显示

### 全部数据模式（散点图）
- **X轴**：测量点序号（标签隐藏，避免拥挤）
- **Y轴**：高程（米）
- **展示方式**：散点图
- **颜色**：根据与预警线的距离
  - 红色：低于预警线
  - 橙色：接近预警线（±1米内）
  - 绿色：正常范围
- **交互**：
  - 悬浮显示海缆名称和点位信息
  - 支持数据缩放（默认显示前20%）
  - 拖拽查看全部数据

## 数据格式要求

### 海缆数据结构
```typescript
interface CableData {
  id: string
  name: string
  type: number // 1=掩埋, 2=裸露
  points: CablePoint[]
}

interface CablePoint {
  lon: number      // 经度
  lat: number      // 纬度
  alt: number      // 高程
  name?: string    // 点位名称（可选）
  raw?: {          // 原始坐标（可选）
    x: number
    y: number
    z: number
  }
}
```

## 后续扩展建议

1. **高程异常检测**：自动识别高程异常的测量点
2. **历史对比**：支持同一海缆不同时间的高程数据对比
3. **报警通知**：当高程低于预警线时发送通知
4. **批量导出**：支持导出多条海缆的分析报告
5. **3D可视化**：在 3D 场景中展示高程变化

## 注意事项

1. 确保海缆数据包含有效的 `points` 数组
2. 高程数据（`alt`）建议使用海平面高程
3. 预警阈值应根据实际业务需求设置
4. 大量数据时建议使用"全部数据"模式的缩放功能
5. 导出的 CSV 文件编码为 UTF-8
