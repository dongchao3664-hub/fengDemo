## 🎯 使用步骤

### 方式 1：从信息面板触发
1. 选择海缆 → 2. 查看详情 → 3. 点击"高程分析"

### 方式 2：从列表快捷触发
1. 点击列表项右侧的 📈 图标 → 直接打开图表

### 方式 3：从地图点击触发
1. 点击地图上的海缆线 → 2. 点击信息面板"高程分析"

## 🎨 颜色说明

### 单条线段模式
- 🟢 **绿色**：掩埋海缆
- 🟠 **橙色**：裸露海缆
- 🔴 **红色虚线**：预警线

### 全部数据模式
- 🔴 **红色点**：低于预警线（需要关注）
- 🟠 **橙色点**：接近预警线（±1米内）
- 🟢 **绿色点**：正常范围

## ⚙️ 功能操作

### 切换视图
- **当前线段**：查看单条海缆的高程趋势（折线图）
- **全部数据**：对比所有海缆的高程分布（散点图）

### 调整预警阈值
- 在"预警阈值"输入框中修改数值
- 图表会实时更新预警线位置
- 全部数据模式下，点的颜色会根据新阈值自动调整

### 导出数据
- 点击"导出数据"按钮
- 自动生成 CSV 文件
- 包含：海缆名称、点位名称、经纬度、高程

## 💡 使用技巧

1. **快速分析**：使用列表中的 📈 图标快速打开图表
2. **全局对比**：切换到"全部数据"模式查看所有海缆
3. **缩放查看**：拖动图表底部的滑块缩放数据范围
4. **详细信息**：鼠标悬浮在图表上查看点的详细信息
5. **数据导出**：定期导出数据用于离线分析

## 🔧 技术细节

### 数据来源
- 从 `mockdata.ts` 中的 `cableData` 加载
- 每条海缆包含多个测量点
- 每个点包含经纬度和高程信息

### 图表类型
- **折线图**（单线段）：echarts line chart with area style
- **散点图**（全部数据）：echarts scatter chart

### 响应式
- 图表自动适应窗口大小变化
- 移动端友好的触摸交互

## 🎓 应用场景

1. **定期巡检**：查看海缆高程变化趋势
2. **异常检测**：识别低于预警线的测量点
3. **维护计划**：根据高程数据制定维护计划
4. **数据分析**：导出数据进行深度分析
5. **报告生成**：为管理层提供可视化报告
